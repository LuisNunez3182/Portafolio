## 2. Implementación Técnica
Para este proyecto, apliqué todas las cosas vistas en clase, integrando además alguna cosas adicionales para hacerlo un poco más interesante:

*   **Gestión de Datos:** Implementé un sistema basado en archivos JavaScript (`.js`) para almacenar la información de proyectos y certificados. Esto facilita enormemente agregar o quitar contenido sin tocar la estructura HTML.
    *   *Nota:* Intenté inicialmente usar archivos `.txt` o `.json`, pero opté por JS para evitar errores de lectura (CORS) al ejecutarlo localmente y simplificar la integración.
*   **Renderizado Dinámico:** Utilicé métodos de array como `map` para generar las secciones del HTML de forma dinámica a partir de los datos.
*   **Control de Errores:** Se implementaron validaciones para asegurar que las secciones carguen aunque los datos fallen en leerse.
